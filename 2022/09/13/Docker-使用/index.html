Docker 使用

安装

sudo pacman -S docker

docker info 报错

Server:

ERROR: Got permission denied while trying to connect to the Docker daemon socket at unix:///var/run/docker.

sock: Get "http://%2Fvar%2Frun%2Fdocker.sock/v1.24/info": dial unix /var/run/docker.sock: connect: permissi

on denied

errors pretty printing info

解决

sudo chmod 666 /var/run/docker.sock

用法

拉取镜像
docker pull mysql
查看镜像
docker images
列出容器
docker ps
运行容器
    docker run -d(指定后台运行) -p(指定端口) 80（要指定的端口）：80 --name mynginx nginx(接镜像名)
进入容器
-it (阻塞运行)
docker exec -it c5(容器id) bash
修改nginx页面
cd /etc/share/nginx/html
echo hello > index.html
退出docker
exit
提交容器
docker commit c5 m

删除容器
docker rm -f (容器id)

vim Dockerfile

构建镜像
docker build -t m2(镜像名) .
运行镜像
docker run -d -p100:80 m2
保存文件
docker save m2>1.tar
加载镜像
docker load < 1.tar
